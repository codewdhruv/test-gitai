# Ludo Game

A modern, interactive implementation of the classic Ludo board game built with HTML5, CSS3, and Vanilla JavaScript.

## 🎮 Features

- **4-Player Gameplay**: Play with up to 4 players (Red, Green, Yellow, Blue)
- **Dice Rolling**: Click the dice to roll and progress pieces
- **Visual Board**: 15x15 grid-based game board with distinct zones
- **Interactive Pieces**: Hover effects and animations for better UX
- **Player Tracking**: Real-time player turn indicator
- **Responsive Design**: Works on different screen sizes
- **Dark Theme**: Modern dark gradient UI

## 🚀 How to Play

1. Open `index.html` in a web browser
2. Players take turns rolling the dice
3. Move your pieces around the board according to dice value
4. First player to get all pieces to home wins!

## 📋 Game Board Layout

- **Home Zones**: Corner areas where pieces start (colored by player)
- **Start Positions**: Entry points to the main path
- **Safe Zones**: Marked with stars, protected from captures
- **Center**: Meeting point of all player paths

## 🛠️ Technical Stack

- **HTML5**: Semantic markup and structure
- **CSS3**: Grid layout, gradients, and animations
- **JavaScript**: Game logic and DOM manipulation
- **No Dependencies**: Pure vanilla JavaScript implementation
